Multilingual Sentiment Analysis
French / English

In my multilingual sentiment analysis project, I utilized XLM-Roberta, a powerful language model, to develop a system capable of accurately analyzing sentiment across multiple languages. By fine-tuning the model on a diverse dataset, I created a solution that can effectively classify sentiment in different languages. The project involved preprocessing, training, and evaluation steps, incorporating transfer learning and addressing language-specific nuances. Ultimately, the project aimed to provide businesses and researchers with a robust tool for gaining insights from text data in various linguistic contexts.
